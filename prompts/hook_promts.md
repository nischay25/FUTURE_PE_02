\# Hook Prompt Documentation



\## Prompt Used

Generate 10 UGC-style hooks for an oversized streetwear clothing brand targeting Gen Z students.



Tone: trendy, engaging, casual, scroll-stopping.



\## Purpose

This prompt was used to create attention-grabbing opening lines for short-form social media advertisements.



\## Expected Output

\- Scroll-stopping hooks

\- Gen Z fashion-focused content

\- Trendy and relatable openings
