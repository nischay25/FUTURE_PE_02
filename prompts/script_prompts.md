\# Script Prompt Documentation



\## Prompt Used

Write 5 UGC ad scripts for a streetwear clothing brand.



Format:

\- Hook

\- Problem

\- Solution

\- Result

\- CTA



Platform: Instagram Reels and TikTok.

Tone: natural and trendy.



\## Purpose

This prompt was used to generate realistic UGC-style advertisement scripts for social media platforms.



\## Expected Output

\- Conversational ad scripts

\- Social media friendly format

\- Fashion marketing content
